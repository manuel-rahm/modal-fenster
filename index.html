Modales Fenster

Dies ist ein modales Fenster, welches Sie mit einem Klick auf "Okay" schliessen können. Öffnen können Sie es wieder mit "Öffnen"

Hier noch ein Bild, damit es ein bisschen besser aussieht.

Was ist es und für was ist es gut?

Als modales Fenster wird jeglicher Inhalt bezeichnet, der vor dem eigentlichen Inhalt der Seite erscheint. Ein solches Fenster muss erst geschlossen werden, bevor der Benutzer mit dem Rest der Seite weiterarbeiten kann.

Das modale Fenster auf dieser Seite wurde sehr aufwändig und von Null auf erstellt. Es gibt einfachere und bessere Wege, um ein modales Fenster zu erstellen.

Für modale Fenster können Sie Bootstrap oder andere Frameworks wie jQuery Modal verwenden.

Ich empfehle Ihnen jQuery Modal, da es einfach zu stylen ist. Es braucht auch keine langen Klassen oder IDs.
