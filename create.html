<h1>Was ist es und für was ist es gut?</h1>
<p class="normal-text">Als modales Fenster wird jeglicher Inhalt bezeichnet, der vor dem eigentlichen Inhalt der Seite erscheint. Ein solches Fenster muss erst geschlossen werden, bevor der Benutzer
mit dem Rest der Seite weiterarbeiten kann.
</p>
<p class="normal-text">Die modalen Fenster auf dieser Webseite wurden alle sehr aufwändig und von Null auf erstellt. Es gibt einfachere und bessere Wege, um ein modales Fenster zu erstellen.</p>
<p class="normal-text">Für modale Fenster können Sie <a href="https://getbootstrap.com/">Bootstrap</a> oder andere Frameworks wie <a href="https://jquerymodal.com/">jQuery Modal</a> verwenden.</p>
<p class="normal-text">Ich empfehle Ihnen jQuery Modal, da es einfach zu stylen ist. Es braucht auch keine langen Klassen oder IDs.</p>

Javascript/jQuery

$('.modal_close').on('click', function(){
$('.modal_content').hide();
$('.modal_close').hide();
$('.modal_open').show();
$('body:not(.modal)').css("background-color", "white").css("opacity", "1");
});

$('.modal_open').on('click', function(){
$('.modal_content').show();
$('.modal_close').show();
$('.modal_open').hide();
$('body').css("background-color", "darkgrey").css("opacity", "0.8");
$(".modal-content").css("opacity", "1");
});
$(function() {
$(".modal_content").dialog({
modal: true,
draggable: false,
resizable: false,
buttons: {
Ok: function() {
$( this ).dialog( "close" );
}
}
});
});
